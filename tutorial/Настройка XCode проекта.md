
### Установка иконки

Создайте архив для иконки через https://www.appicon.co/

распакуйте архив и перенесите содержимое архива в <папка проекта>/ios/Runner и замените файлы

### Замена бандла в проекте 

В VS Code  нажмите Shift + Command + H, замените везде значение 
```
com.lucky.routine
```
на Bundle ID вашего проекта
