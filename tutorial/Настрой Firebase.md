### Что нужно дополнительно сделать

В первый раз нужно добавить Firebase CLI
```
curl -sL https://firebase.tools | bash
```

Войти в свою рабочую учетку 
```
firebase login
```

#### Всегда при настройке проекта 
1.  Добавить в Firebase проект свою рабочую почту

Из любого каталога выполните эту команду:

```
dart pub global activate flutterfire_cli
```

Затем в корневом каталоге вашего проекта Flutter выполните следующую команду:
**!Обязательно заменить на ВАШ bundle ID**

```
flutterfire configure --ios-bundle-id <com.yourcompany.yourapp>
```

Это автоматически зарегистрирует ваши платформенные приложения в Firebase и добавит `lib/firebase_options.dart`файл конфигурации в ваш проект Flutter.

Нужно будет настроить при подтягивании
